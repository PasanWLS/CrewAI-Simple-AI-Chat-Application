Simple AI Chat Application Using CrewAI and Gemini

## Steps -

1. Create New Project Folder 'CrewAI'

2. Create Virtual Environment
Terminal > pip install virtualenv > virtualenv env > env\Scripts\activate

3. Install Libraries
pip install -r requirements.txt

## Libraries -  
crewai ,
crewai-tools

## .env file
GEMINI_API_KEY=your_gemini_api_key
